Deployment Guide

Full step-by-step deployment instructions are included in DEPLOY.md covering:

  • Step 1: Set up OnionBalance manager on a dedicated server
  • Step 2: Configure setup.sh with your backend connection and encryption keys
  • Step 3: Customize branding (logo, favicon, colors)
  • Step 4: Get dependencies (run once)
  • Step 5: Deploy to 3+ front servers (Debian 10)
  • Step 6: Link all fronts to OnionBalance
  • Step 7: Validate with the pentest toolkit
  • Step 8: Tune rate limits and WAF for your application

⚠️ Server Requirements

Deployment requires dedicated Debian 10 servers with root access. Minimum: 1 manager (2 CPU / 2 GB RAM) + 3 front servers (4 CPU @ 3GHz+ / 4 GB RAM each).
